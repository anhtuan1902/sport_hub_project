## Tб»•ng Quan

### Database Engine
- **PostgreSQL 15+** vб»ӣi cГЎc extensions:
  - `uuid-ossp` - UUID generation
  - `postgis` - Geospatial queries
  - `pg_trgm` - Full-text search

### Naming Conventions

| LoбәЎi | Quy TбәҜc | VГӯ dб»Ҙ |
|------|----------|--------|
| Tables | snake_case, plural | `users`, `court_bookings` |
| Columns | snake_case | `created_at`, `full_name` |
| Primary Keys | `id` (UUID) | UUID v4 |
| Foreign Keys | `<table>_id` | `user_id`, `court_id` |
| Indexes | `idx_<table>_<column>` | `idx_users_email` |
| Enums | snake_case | `user_status` |

## Backup & Restore

### Backup

```bash
# Full database backup
pg_dump -h localhost -U postgres -d sport_hub > backup_$(date +%Y%m%d).sql

# Backup with compression
pg_dump -h localhost -U postgres -d sport_hub | gzip > backup_$(date +%Y%m%d).sql.gz
```

### Restore

```bash
# Restore from backup
psql -h localhost -U postgres -d sport_hub < backup_20260509.sql

# Restore with compression
gunzip -c backup_20260509.sql.gz | psql -h localhost -U postgres -d sport_hub
```
